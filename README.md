## Run Locally

Clone the project

```bash
  git clone https://github.com/satya-ranjon/Kanban_board
```

Go to the project directory

```bash
  cd Kanban_board
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
